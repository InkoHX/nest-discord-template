# nest-discordjs-template

![Node.js CI](https://github.com/InkoHX/nest-discord-commander/workflows/Node.js%20CI/badge.svg)

## Description

This is a template for using Discord.js with Nest.js.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
